# NEU Library Visitor Management System

A digital library visitor logbook for New Era University.

## Live Application
https://neu-library-app-eb94f.web.app

## GitHub Repository
https://github.com/marcuspaguia-lgtm/neu-library-app

## Features
- Google Authentication (NEU institutional emails only)
- Library check-in system
- Admin dashboard with visitor statistics
- Role-based access control
